# Telegram Reminder Bot

Простой бот для Telegram-групп, который ставит напоминания и упоминает выбранного участника.

## Возможности
- `/remind 10m Текст` — напоминание без адресата.
- `/remind @username 10m Текст` — напоминание на указанного участника.
- Ответом на сообщение: `/remind 10m Текст` — цель определяется по реплаю.
- `/list` — показать активные напоминания в чате.
- `/cancel <id>` — отменить напоминание по ID из `/list`.

Поддерживаемые форматы времени: `10m`, `1h20m`, `2d3h`, `45s` (s, m, h, d).

## Запуск
1. Установите зависимости:
   ```bash
   python3 -m pip install -r requirements.txt
   ```
2. Задайте токен бота от `@BotFather`:
   ```bash
   export BOT_TOKEN="ваш_токен"
   ```
3. Запустите:
   ```bash
   python3 bot.py
   ```

Бот использует polling. Добавьте его в группу и выдайте права на чтение сообщений. Для напоминаний по реплаю ему нужно видеть, кто автор сообщения.

## Примеры
- Ответом на сообщение: `/remind 20m Проверь задачу`
- В явном виде: `/remind @alex 1h30m Пройтись по PR`
- Список: `/list`
- Отмена: `/cancel <id>` (ID берётся из `/list` или ответа бота при постановке).
